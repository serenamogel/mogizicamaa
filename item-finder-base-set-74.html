Item Finder (Japanese: ダウジングマシーン Dowsing Machine) is a Trainer card. It was first released as part of the Base Set.

Card text

Discard 2 of the other cards from your hand in order to put a Trainer card from your discard pile into your hand.

Release information

This card was included in the Base Set, first released in the Japanese Expansion Pack. It was reprinted in the Base Set 2 expansion.

Gallery

Trivia

  • This card's Japanese and English names are both the same as the item in the games, but spelled differently. The Japanese card is named ダウジングマシーン as opposed to ダウジングマシン, and the English card is named Item Finder as opposed to Itemfinder. The card Dowsing Machine was later released with the same effect as this card and the same name as the item in the games.
  • This card's effect is similar to Junk Arm, released in the Triumphant expansion.
